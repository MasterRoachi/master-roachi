# Master Roachi Hub

This is the first version of the Master Roachi website.

It was built as my replacement for The Odin Project Recipes project.

## Purpose

The purpose of this project is to practice basic HTML by creating a complete multi-page website.

The website acts as the central hub for the Master Roachi project and connects three branches:

- Arkitecture
- Questicles
- South African Sinner
